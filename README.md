# ゲーム説明
簡単に、ゲームの一連の流れを説明します。
主人公の「マルチタスカー」君は、「世界の真実」を追う旅の中で山奥に辿り着きます。
しかし、その山奥にはすでに町が存在し、ダンジョンにある「世界の真実」を求めて多くの冒険者が集まっていました。

ダンジョン内は不思議な力に満たされており、特殊な敵や超能力が使えるようになる「カード」や便利な装備物品等が手に入ります。
ダンジョンに潜入->カードや装備を入手->強くなる->ダンジョンに潜入
というループを繰り返すことになります。
この間、クエストの発生や、町での売買などのコンテンツがあります。

戦闘は、カードを敵や自身にドラッグすることで行われます。
毎ターン3得られるAP(アクションポイント)を使用し、カードを用いて防御したり、敵に攻撃をすることができます。
カードの特徴として、「念動」や「エネルギー」などの、超能力をモチーフとした属性を持っていることが挙げられます。
この属性は、ほかのカード効果によって参照されます。
例えば、「エネルギーコスト減少」を得る効果のカードを使用すると、次に使用する「エネルギー」カードのコストが1減少します。

また、「武器」という要素も存在します。
武器は、「いつでも」、「何度でも」使用できるカードとしてふるまいます。
敵を倒したときにドロップし、装備画面で変更可能です。
一部に特殊な効果の武器も存在します。
